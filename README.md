# datagenda
Datagenda, um micro-ERP de agendamento online voltado para clínicas e empresas de
pequeno porte.
A solução está sendo desenvolvida com JavaScript, Node.js, PostgreSQL, HTML,
Bootstrap e React, contemplando uma arquitetura moderna e escalável.
O principal objetivo do sistema é oferecer um gerenciamento completo das operações
empresariais, incluindo controle financeiro, gestão de produtos e estoque, vendas e
agendamentos online de serviços, proporcionando uma experiência integrada e eficiente
para os usuários.

Utilizado Knex.js para integração de dados com Postgres.

Express para rotas de comunicação.

Axios.js para contralor todas as requisições entre Front e back.

Execução;

-Configurar Postgres conforme .env da raiz do projeto

-Executar migrations

-Executar start.bat para iniciar o servidor node.js
